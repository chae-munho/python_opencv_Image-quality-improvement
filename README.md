# 🖼️ OpenCV Installation Guide

이 프로젝트를 실행하기 전에 필요한 라이브러리를 설치하세요.

## 📦 Installation

```bash
# 기본 OpenCV 설치
pip install opencv-python

# 추가 모듈 (SIFT, SURF 등 고급 기능 필요시)
pip install opencv-contrib-python

# pip 최신 버전으로 업데이트 (권장)
python -m pip install --upgrade pip
